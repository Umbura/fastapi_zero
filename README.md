The idea behind this course is not only to learn FastAPI, but to build a complete Python ecosystem for web development without AI assistance.
Topics covered include:

Web development, FastAPI, Pydantic, Uvicorn, SQLAlchemy, JWT, asyncio

Week 1: Setting up the development environment
Week 2: Introduction to web development
Week 3: Structuring the project and creating CRUD routes
Week 4: Configuring the database and managing migrations with Alembic
Week 5: Integrating the database with the API
Week 6: Authentication and Authorization with JWT
Week 7: Refactoring the project structure
Week 8: Making the project asynchronous
Week 9: Making the authentication system robust
Week 10: Creating CRUD routes for task management
Week 11: Dockerizing our application and introducing PostgreSQL
Week 12: Automating tests with Continuous Integration
Week 13: Deploying to Fly.io
Week 14: Farewell and next steps
Week 15: Final project
